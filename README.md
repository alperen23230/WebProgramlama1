# Dosyalar

1. pages/index.js => Twitter sitesindeki Home (Anasayfa) sayfasının kopyasıdır.
2. pages/profile.js => Twitter sitesindeki Profile (Hakkımda)'dır.
3. pages/more.js => Twitter sitesindeki More sayfası tema seçimi yapılabilmesi için gerekli olan sayfadır.
4. components/* => Sitedeki sayfalarda görünen herşey bu dizindeki component'ler kullanılarak yapılmıştır.
